# Mail failover

## change credentials
### copy .env.sample to .env
### update your Mailgun and Sendgrid credentials

## Install global package
### 1. npm install -g vue-cli
### 2. npm install -g quasar-cli

## Install client side package
### 1. cd client
### 2. npm install
### 3. quasar build

## Install server side package
### 1. cd server
### 2. npm install
### 3. node index.js
